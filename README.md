## 🔍 About Us

**Flood Agent**는 기후 변화로 인한 수해 및 재난 문제를 AI 기술로 해결하고자 모인 팀입니다.  
우리는 실시간 홍수 예측, 피해 경로 시뮬레이션, 대응 전략 자동화 등 다양한 관점에서  
AI 기반의 재난 대응 시스템을 개발하고자 합니다.

- **팀명:** Flood Agent

## 🎯 프로젝트 목표

> "AI로 생명을 구하고, 안전을 설계한다."

- 홍수 예측을 위한 시계열/공간 데이터 기반 모델 개발
